# Security Policy

Monad is currently pre-public-release software.

## Supported versions

| Version | Supported |
|---|---|
| Internal MVP candidate | Best-effort internal review |
| Public pre-release | Not yet released |

## Reporting a vulnerability

Do not disclose security vulnerabilities in public issues.

Until a dedicated security contact is finalized, report suspected vulnerabilities privately to:

```text
thomas.carter@appliedinnovationcorp.com
```

Include:

* affected component
* reproduction steps
* impact
* affected version or commit
* whether the issue is publicly known

## Security posture

High or critical dependency alerts block public pre-release unless they are resolved or formally risk-accepted.

Generated/imported artifacts must not create false first-party dependency surfaces.
