# Monad Context Artifacts

This directory is reserved for Monad-generated or Monad-maintained context artifacts.

Future examples may include:

```text
current-state.md
latest-handoff.md
latest-context-pack.md
session-chronicles/
decision-log.md
```

Generated context must distinguish facts, accepted decisions, drafts, assumptions, and inferred summaries.
