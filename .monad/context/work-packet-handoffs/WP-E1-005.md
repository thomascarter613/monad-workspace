## Objective

Create a durable manifest model foundation in `monad-core`.

## Manifest Model

This slice adds:

* root `monad.toml`;
* `ManifestSchemaVersion`;
* `ManifestProject`;
* `ManifestWorkspace`;
* `ManifestRuntime`;
* `MonadManifest`;
* manifest diagnostics;
* manifest validation;
* runtime identity integration.

## Important Boundary

This slice does not parse TOML yet.

Parsing and file loading should be introduced in the next slice after the model is stable.

## Rules
