# Session Chronicles

## Purpose

This directory is reserved for session chronicle files.

A session chronicle records what happened during a work session:

- starting state;
- active work packet;
- files changed;
- verification run;
- commit made;
- next recommended action.

## Naming

Recommended future naming:

```text
YYYY-MM-DD-session-NNN.md
```

Example:

```text
2026-05-23-session-001.md
```
