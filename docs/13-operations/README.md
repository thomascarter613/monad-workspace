# Operations

## Purpose

This directory defines how Monad is set up, maintained, released, and supported.

## Belongs Here

* Local development.
* Repository setup.
* Toolchain setup.
* Release process.
* Versioning policy.
* Support model.
* Maintenance model.
* Backup and exports.

## Does Not Belong Here

* Product vision.
* Architecture decisions.
* Generated evidence packets.
