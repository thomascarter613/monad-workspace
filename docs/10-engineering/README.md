# Engineering

## Purpose

This directory defines Monad engineering standards.

## Belongs Here

* Coding standard.
* Rust coding standard.
* Rust learning notes.
* Rust verification.
* Error handling standard.
* Diagnostic standard.
* Testing standard.
* Fixture standard.
* Output format standard.
* CLI UX standard.
* Dependency standard.

## Does Not Belong Here

* Product strategy.
* Business strategy.
* GitHub project board configuration.
* Generated reports.
