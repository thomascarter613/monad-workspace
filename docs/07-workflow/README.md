# Workflow

## Purpose

This directory defines how Monad work is planned, executed, verified, reviewed, committed, and handed off.

## Belongs Here

* Operating model.
* Work hierarchy.
* Epic standard.
* Work packet standard.
* Task standard.
* Deliverable standard.
* Definition of Ready.
* Definition of Done.
* Verification standard.
* Commit standard.
* Branching standard.
* Review standard.
* Release standard.
* Context update standard.

## Does Not Belong Here

* Product vision.
* Domain model.
* Implementation code.
* Generated reports.
